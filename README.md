# Homework bot
## Описание проеткта
Проект телеграм бота, который проверяет статус проверки домашней работы на платформе Яндекс.Практикум. При запуске программы к API платформы отправляется запрос, в котором передается текущая дата, время и уникальный для каждого студента токен авторизации. С промежутком в 10 минут запрос повторяется, если со времени последнего запроса статус работы изменился, в ответ приходит не пустой обект json. 
Пример ответа сервера:
```
{
   "homeworks":[
      {
         "id":124,
         "status":"rejected",
         "homework_name":"username__hw_python_oop.zip",
         "reviewer_comment":"Код не по PEP8, нужно исправить",
         "date_updated":"2020-02-13T16:42:47Z",
         "lesson_name":"Итоговый проект"
      },
      {
         "id":123,
         "status":"approved",
         "homework_name":"username__hw_test.zip",
         "reviewer_comment":"Всё нравится",
         "date_updated":"2020-02-11T14:40:57Z",
         "lesson_name":"Тестовый проект"
      },

      ...

   ],
   "current_date":1581604970
} 
```
Статус работы отправляется пользователю в телеграмм бота. Если в процессе работы программы возникают исключения, вся информация логируется и передается в stdout. Все ошибки так же дублируются пользователю в телеграмм бота однократно. Важные токены: PRACTICUM_TOKEN, TELEGRAM_TOKEN, TELEGRAM_CHAT_ID сохраняются в корне директории в файле .env 

## Технологии
- python 3.9 - язык программирования, на котором написан проект.
- telegram - библиотека для работы с API телеграмм.
- logging - встроеная библиотека языка для логирования.
- requests - библиотека для отправки запросов на удаленный сервер.
- dotenv - инструмент для создания переменных окружения из файла .env

## Запуск программы
Для использования программы нужно установить интерпретатор языка Python. Дополнительные зависимости устанавливаются из файла requirements.txt, для этого нужно прописать в командной строке 
```
pip install -r requirements.txt. 
```
Для работы также потребуется создать в дириктории программы файл .env, в котором нужно прописать следующие переменные:
```
PRACTICUM_TOKEN=
TELEGRAM_TOKEN=
TELEGRAM_CHAT_ID=
```
Запуск программы происходит через файл 'homework.py'
